Utilities¶

Common utility functions and helpers.

server.utils.dataset_hash(segments: list[SegmentPostData], source_lang, target_lang) → str[source]¶

Calculate the hash of the dataset. We canonicalize the JSON representation. Currently use blake2b as the hashing algorithm.

For canonical JSON, see JCS(RFC8785): - https://datatracker.ietf.org/doc/html/rfc8785

Note that we take only the source and reference segments into account as the target segments are linked to a run, not a dataset.
